# EventosPro App

### Descripción breve
**EventosPro App** es una solución integral de escritorio diseñada para la planificación y administración de eventos. La aplicación permite centralizar la gestión de recursos, personal y sedes a través de una interfaz robusta, garantizando un control detallado de cada etapa del proceso organizativo.

---

### 4. MÓDULOS DEL SISTEMA
El sistema se encuentra dividido en los siguientes componentes funcionales:

* **Módulo de Acceso al Sistema:** Control de seguridad y validación de usuarios.
* **Módulo de Gestión de Clientes:** Administración de la base de datos de clientes y sus requerimientos.
* **Módulo de Gestión de Eventos:** Núcleo del sistema para la creación y seguimiento de eventos.
* **Módulo de Gestión de Sedes:** Registro y control de disponibilidad de espacios físicos.
* **Módulo de Coordinadores:** Gestión del personal encargado de la supervisión de eventos.
* **Módulo de Proveedores:** Directorio y administración de servicios externos contratados.
* **Módulo de Dashboard:** Panel visual con indicadores clave y resumen de actividades.
